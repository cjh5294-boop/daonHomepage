## 개발 히스토리

### 1단계 - 초기 시안 제작
- PPT(`공동방제 제안 PPT.pptx`) 기반으로 디자인 방향 설정
- 총 3가지 타입 시안 제작:
  - `type1.html` - 기본 시안
  - `type2.html` - 딥네이비 + 스카이블루 컬러
  - `type3.html` - 클린 화이트 모던 (채택 기준)
- `index.html` - 기본 랜딩

### 2단계 - 구조 시안 제작 (struct1.html)
- `type3.html` 기준으로 전체 구성 단계 HTML 제작
- 포함 섹션:
  - NAV: 로고 + 메뉴 (사업분야/영상갤러리/공지사항/문의) + 햄버거(모바일)
  - HERO: 슬로건 좌측 + 홍보영상 플레이스홀더 우측
  - 사업분야: 4개 카드 (자동방제/드론교육/드론운송/DJI판매)
  - 영상갤러리: YouTube 영상 카드 3개 + 더보기 버튼
  - 공지사항: 게시판 형태 리스트 (NEW 배지)
  - 문의하기: 연락처 + 문의 폼
  - Footer: 회사 정보
- 조박사님이 직접 수정 후 Git push → 로컬 pull 반영 완료
