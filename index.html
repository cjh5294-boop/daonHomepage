Problem

수동방제, 이대로 괜찮을까요?

현재 보편화된 수동방제 방식은 다양한 문제를 안고 있습니다.

⚖️

농약 불균형 살포

작업자의 숙련도에 따라 농약 살포량이 달라져 과다 또는 과소 살포가 반복됩니다.

🧑‍🌾

인력 부족 & 고령화

농촌 인구 감소와 고령화로 방제 인력 확보가 점점 어려워지고 있습니다.

⏱️

낮은 작업 효율

넓은 면적을 수동으로 방제하면 시간과 비용이 과도하게 소요됩니다.

☣️

농약 노출 위험

작업자가 직접 농약을 살포해 건강에 심각한 위협이 될 수 있습니다.

Solution

다온이 제시하는
자동방제 솔루션

RTK 정밀 측위 기반 드론 자동방제로 모든 문제를 해결합니다.

🚁
  • ✓

    정밀 균일 살포
    RTK GPS 기반으로 cm 단위 정확도의 균일한 농약 살포를 실현합니다.

  • ✓

    광역 신속 방제
    드론 15대 동시 운용으로 대규모 농지를 3일 내 완료합니다.

  • ✓

    완전 자동화
    GIS 데이터 기반 자동 경로 설정으로 작업자 개입 없이 방제합니다.

  • ✓

    안전한 무인 방제
    작업자 농약 노출 없이 원격 모니터링으로 안전하게 운용합니다.

FLEET
Process

방제 진행 프로세스

체계적인 5단계 프로세스로 완벽한 방제를 보장합니다.

01

사전 접수

농지 정보 및 방제 일정 접수, 현장 조건 사전 파악

02

항공 측량

RTK 드론으로 농지 정밀 측량, 3D 지형 데이터 수집

03

GIS 확인

공간 데이터 분석 및 최적 방제 경로·살포량 자동 설정

04

자동 방제

드론 군집 자동 운용, 실시간 원격 모니터링

05

결과 보고

방제 완료 리포트 제공, 살포 이력 데이터 기록
