## 문제 정의 (홈페이지에서 강조할 Pain Point)
- 수동방제가 현재 보편화되어 있음
- 수동방제의 문제: **농약 불균형 살포**, 노동력 부족, 비용 과다, 품질 불균일
- 고령화로 인한 방제 인력 부족

## 다온의 솔루션 (강점)
1. **RTK 정밀 자동방제** - cm급 오차의 정밀 비행
2. **GIS 기반 농지 데이터** - 지리정보시스템으로 정확한 면적 산출
3. **드론 15대 + 17명** - 3일 이내 광역 방제 가능
4. **방제 보조프로그램(지오팜)** - 실시간 방제 정보 공유
5. **항공 측량** - 정확한 항공지도 제작

## 홈페이지 구성안 (초안)

### 페이지 구조
```
/ (메인)
├── #hero        - 임팩트 있는 슬로건 + 드론 영상/이미지
├── #problem     - 수동방제의 문제점
├── #solution    - 다온의 자동방제 솔루션
├── #technology  - RTK / GIS / 방제 프로그램 기술 소개
├── #process     - 방제 프로세스 (사전→방제→사후)
├── #results     - 실적 및 기대효과
├── #contact     - 문의하기
```

### 디자인 방향
- **컬러**: 녹색 계열(농업) + 파란 하늘(드론) + 화이트
- **톤**: 신뢰감 있는 전문기업 이미지
- **레이아웃**: 풀스크린 히어로 + 스크롤 애니메이션
- **반응형**: 모바일 우선
